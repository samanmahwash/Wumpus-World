# Wumpus World Knowledge-Based Agent

A web-based AI agent that navigates a Wumpus World grid using propositional logic and resolution-based inference.

## Features
- Dynamic grid size (rows × columns)
- Random placement of pits and Wumpus
- Breeze and stench percept generation
- Knowledge Base using propositional logic
- Resolution refutation for safety checking
- Real-time inference log
- Visual grid with agent movement

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## How to Run
Simply open `index.html` in any browser.
